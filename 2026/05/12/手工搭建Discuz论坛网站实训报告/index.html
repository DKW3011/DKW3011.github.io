手工搭建Discuz论坛网站实训报告

一、实训目的

本次实训以手工部署的方式搭建 Discuz 论坛网站，核心目标如下：

  1. 掌握在 CentOS Linux 环境下，使用命令行完成 MySQL 数据库的安装、配置与权限管理的方法。

  2. 理解并搭建 LAMP（Apache+MySQL+PHP）Web 运行环境，掌握 Web 服务的启动、自启配置与状态验证流程。

  3. 掌握 Discuz 论坛系统的代码部署、安装向导配置与网站验证的完整流程。

  4. 理解基于华为云弹性云服务器的网站整体架构，对比镜像部署与手工部署的差异，了解分布式部署（数据库与 Web 服务分离）的设计逻辑。

二、实训环境

环境类型 具体配置
云平台 华为云弹性云服务器 ECS（实验沙箱环境）
操作系统 CentOS
软件组件 MySQL 数据库、Apache Web 服务器、PHP 运行环境、Discuz! X3.4 简体中文版
操作工具 Chrome 浏览器（无痕模式）、华为云 CloudShell 远程登录工具

三、实训内容

本次实训围绕 “手工搭建 Discuz 论坛” 展开，核心内容分为四大模块：

  1. 实验环境准备：登录华为云实验控制台，获取实验资源与操作权限。

  2. 数据库服务器搭建：在 discuz01 节点完成 MySQL 的安装、初始化、权限配置与防火墙设置，为论坛提供数据存储支持。

  3. Web 环境搭建：在 discuz02 节点完成 Apache、PHP 环境的安装配置，通过安全组与防火墙配置开放 Web 访问端口，并验证 Web 服务可用性。

  4. Discuz 论坛部署与验证：下载并部署 Discuz 代码包，配置目录权限，通过安装向导完成论坛初始化，最终验证网站可正常访问。

  2. 确认实验资源已分配，获取弹性云服务器 discuz01、discuz02 的登录信息与 IP 地址。

阶段五：实验收尾

实验完成后，返回华为云实验手册页面，点击 “结束实验”，释放实验环境资源，完成本次实训。

五、实训结论

  1. 实训成果
    本次实训成功完成了基于华为云的 Discuz 论坛手工搭建，实现了数据库服务器与 Web 服务器的分离部署：MySQL 数据库服务运行正常，可支持远程连接；LAMP Web 环境搭建完成，Apache 服务可正常响应请求；Discuz 论坛安装配置完成，可通过公网 IP 正常访问，实现了论坛的基础功能。

  2. 知识与技能收获

    • 掌握了 CentOS 系统下 MySQL 数据库的安装、初始化、远程访问配置流程，理解了数据库权限管理的核心逻辑。

    • 理解了 LAMP 架构的工作原理，掌握了 Apache、PHP 服务的启动、自启配置与故障排查方法。

    • 熟悉了 Web 应用手工部署的完整流程，对比镜像部署，手工部署更深入地理解了各组件的依赖关系与配置要点。

  3. 问题与解决经验

    • 权限问题：Discuz 安装过程中因目录权限不足报错，通过chmod \-R 777 /var/www/html/赋予目录读写权限解决。

    • 访问问题：初始访问 Web 服务器失败，排查发现安全组未放行 80 端口，配置安全组规则后恢复正常访问。

  4. 实训总结
    本次实训让我深入理解了 Web 应用的底层架构与云服务器部署逻辑，掌握了数据库、Web 服务与论坛系统的配置技能，为后续 Web 应用的部署与运维打下了坚实基础。
